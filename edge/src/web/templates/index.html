Development Notes
Important Development Guidelines:
  • All blueprints use absolute imports from v1_3.src.*
  • Services are accessed via get_service('service_name') from dependency container
  • All API responses include timestamp and success fields
  • Error responses include error field with descriptive message
  • Template variables are passed explicitly to avoid undefined errors
  • WebSocket events use room-based communication for real-time updates
  • Database operations are handled through database_manager service
  • Image serving uses Flask routes for security and flexibility
