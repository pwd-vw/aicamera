# คู่มือผู้ใช้ - AI Camera System

## 📋 ภาพรวม

คู่มือผู้ใช้นี้จะแนะนำการใช้งาน AI Camera System ระบบกล้องตรวจจับป้ายทะเบียนรถยนต์แบบอัจฉริยะ ที่พัฒนาด้วยเทคโนโลยีล่าสุดเพื่อการตรวจจับและวิเคราะห์ข้อมูลการจราจรแบบ Real-time

## 🚀 การเริ่มต้นใช้งาน

### การเข้าสู่ระบบ

1. **เปิดเว็บเบราว์เซอร์** และไปยัง URL ของระบบ
2. **กรอกข้อมูลเข้าสู่ระบบ**:
   - Username: ชื่อผู้ใช้
   - Password: รหัสผ่าน
3. **คลิก "เข้าสู่ระบบ"** หรือกด Enter
4. **ระบบจะนำคุณไปยังหน้า Dashboard** หลังจากเข้าสู่ระบบสำเร็จ

### การออกจากระบบ

1. **คลิกที่ชื่อผู้ใช้** ที่มุมขวาบน
2. **เลือก "ออกจากระบบ"**
3. **ยืนยันการออกจากระบบ**

## 🏠 หน้า Dashboard

### ภาพรวม Dashboard

หน้า Dashboard เป็นหน้าหลักของระบบที่แสดงข้อมูลสำคัญต่างๆ:

- **📹 กล้อง**: จำนวนกล้องทั้งหมดและสถานะ
- **🚗 การตรวจจับ**: จำนวนการตรวจจับวันนี้
- **📊 การวิเคราะห์**: สถิติการใช้งาน
- **🔧 สถานะระบบ**: สถานะการทำงานของระบบ

### การนำทาง

#### เมนูหลัก (Header Navigation)
ที่ส่วนบนของหน้าจอจะมีเมนูนำทางหลัก:

- **ชื่อผู้ใช้ (บทบาท)**: แสดงข้อมูลผู้ใช้ปัจจุบัน
- **👥 User Management**: จัดการผู้ใช้ (เฉพาะ Admin)
- **👤 Profile**: จัดการข้อมูลส่วนตัว
- **🚪 Logout**: ออกจากระบบ

#### การ์ดนำทาง (Dashboard Cards)
Dashboard มีการ์ดนำทางไปยังส่วนต่างๆ ของระบบ:

1. **📹 Cameras**: จัดการกล้อง
2. **🚗 Detections**: ดูการตรวจจับ
3. **👥 User Management**: จัดการผู้ใช้ (เฉพาะ Admin)
4. **📊 Analytics**: วิเคราะห์ข้อมูล
5. **📈 Visualizations**: แสดงผลข้อมูล

## 👥 การจัดการผู้ใช้ (User Management)

**หมายเหตุ**: ฟีเจอร์นี้ใช้ได้เฉพาะผู้ใช้ที่มีบทบาท Admin เท่านั้น

### การเข้าถึง User Management

1. **จากเมนูหลัก**: คลิก "👥 User Management" ที่ส่วนบนของหน้าจอ
2. **จาก Dashboard**: คลิกที่การ์ด "👥 User Management"

### การสร้างผู้ใช้ใหม่

1. **กรอกข้อมูลในฟอร์ม "Create New User"**:
   - **Username**: ชื่อผู้ใช้ (ห้ามซ้ำ)
   - **Email**: อีเมล (ห้ามซ้ำ)
   - **First Name**: ชื่อจริง (ไม่บังคับ)
   - **Last Name**: นามสกุล (ไม่บังคับ)
   - **Password**: รหัสผ่าน
   - **Role**: บทบาท (Viewer หรือ Admin)

2. **คลิก "Create User"** เพื่อสร้างผู้ใช้

### การจัดการผู้ใช้

#### ดูรายการผู้ใช้
ตารางแสดงข้อมูลผู้ใช้ทั้งหมด:
- **Username**: ชื่อผู้ใช้
- **Email**: อีเมล
- **Role**: บทบาท (Admin/Viewer)
- **Status**: สถานะ (Active/Inactive)
- **Last Login**: เข้าใช้ครั้งล่าสุด
- **Actions**: ปุ่มจัดการ

#### การเปิด/ปิดใช้งานผู้ใช้
1. **คลิก "Activate"** เพื่อเปิดใช้งานผู้ใช้
2. **คลิก "Deactivate"** เพื่อปิดใช้งานผู้ใช้

#### การลบผู้ใช้
1. **คลิก "Delete"** ที่ผู้ใช้ที่ต้องการลบ
2. **ยืนยันการลบ** ในกล่องข้อความ
3. **หมายเหตุ**: ไม่สามารถลบตัวเองได้

## 👤 การจัดการข้อมูลส่วนตัว (Profile)

### การเข้าถึง Profile

**คลิก "👤 Profile"** ที่เมนูหลักส่วนบนของหน้าจอ

### การแก้ไขข้อมูลส่วนตัว

#### ข้อมูลที่แก้ไขได้:
- **Email**: อีเมล
- **First Name**: ชื่อจริง
- **Last Name**: นามสกุล

#### ข้อมูลที่ดูได้อย่างเดียว:
- **Username**: ชื่อผู้ใช้
- **Role**: บทบาท
- **Account Status**: สถานะบัญชี

#### วิธีการแก้ไข:
1. **แก้ไขข้อมูลในฟอร์ม "Profile Information"**
2. **คลิก "Update Profile"** เพื่อบันทึกการเปลี่ยนแปลง

### การเปลี่ยนรหัสผ่าน

1. **กรอกข้อมูลในส่วน "Change Password"**:
   - **Current Password**: รหัสผ่านปัจจุบัน
   - **New Password**: รหัสผ่านใหม่
   - **Confirm New Password**: ยืนยันรหัสผ่านใหม่

2. **คลิก "Change Password"** เพื่อเปลี่ยนรหัสผ่าน

### ข้อมูลบัญชี

ส่วน "Account Information" แสดงข้อมูล:
- **User ID**: รหัสผู้ใช้ในระบบ
- **Member Since**: วันที่สมัครสมาชิก
- **Last Login**: เข้าใช้ครั้งล่าสุด
- **Profile Updated**: อัปเดตข้อมูลครั้งล่าสุด

## 📹 การจัดการกล้อง

### ดูรายการกล้อง

1. **คลิกที่การ์ด "📹 Cameras"** ในหน้า Dashboard
2. **ระบบจะแสดงรายการกล้องทั้งหมด** พร้อมข้อมูล:
   - ชื่อกล้อง
   - สถานะ (Active, Inactive, Error)
   - ตำแหน่ง
   - จำนวนการตรวจจับ

### การกรองข้อมูล

คุณสามารถกรองข้อมูลกล้องได้:

- **สถานะ**: เลือก Active, Inactive, หรือ Error
- **ค้นหา**: พิมพ์ชื่อกล้องหรือ ID
- **เรียงลำดับ**: คลิกที่หัวตารางเพื่อเรียงลำดับ

### เพิ่มกล้องใหม่

1. **คลิกปุ่ม "Add Camera"** ที่มุมขวาบน
2. **กรอกข้อมูลกล้อง**:
   - Camera ID: รหัสกล้อง (ไม่ซ้ำ)
   - Name: ชื่อกล้อง
   - Location Address: ที่อยู่
   - Latitude/Longitude: พิกัด GPS
   - Image Quality: คุณภาพภาพ (Low, Medium, High)
   - Upload Interval: ช่วงเวลาอัพโหลด (วินาที)
   - Enable Detection: เปิด/ปิดการตรวจจับ
3. **คลิก "Create"** เพื่อบันทึก

### แก้ไขข้อมูลกล้อง

1. **คลิกปุ่ม "Edit"** ในแถวของกล้องที่ต้องการแก้ไข
2. **แก้ไขข้อมูล** ตามต้องการ
3. **คลิก "Update"** เพื่อบันทึกการเปลี่ยนแปลง

### ลบกล้อง

1. **คลิกปุ่ม "Delete"** ในแถวของกล้องที่ต้องการลบ
2. **ยืนยันการลบ** ในหน้าต่างที่ปรากฏ
3. **คลิก "Delete"** เพื่อยืนยัน

### ดูรายละเอียดกล้อง

1. **คลิกปุ่ม "View"** ในแถวของกล้องที่ต้องการดูรายละเอียด
2. **ระบบจะแสดงข้อมูลละเอียด**:
   - ข้อมูลพื้นฐาน
   - สถานะสุขภาพ
   - การตั้งค่า
   - ประวัติการตรวจจับ

## 📱 การใช้งานบนมือถือ

### การตอบสนอง (Responsive Design)

ระบบได้รับการออกแบบให้ใช้งานได้บนทุกอุปกรณ์:

- **Desktop**: แสดงข้อมูลครบถ้วน
- **Tablet**: ปรับขนาดให้เหมาะสม
- **Mobile**: แสดงข้อมูลสำคัญ

### การใช้งานบนมือถือ

- **การนำทาง**: ใช้เมนูแฮมเบอร์เกอร์
- **การกรอง**: ใช้ปุ่มกรองที่ด้านล่าง
- **การดูรายละเอียด**: แตะที่รายการเพื่อดูรายละเอียด

## 🔔 การแจ้งเตือน

### ประเภทการแจ้งเตือน

ระบบมีการแจ้งเตือนหลายประเภท:

- **✅ สำเร็จ**: การดำเนินการสำเร็จ
- **⚠️ เตือน**: ข้อมูลที่ควรระวัง
- **❌ ข้อผิดพลาด**: เกิดข้อผิดพลาด
- **ℹ️ ข้อมูล**: ข้อมูลทั่วไป

### การจัดการการแจ้งเตือน

- **ปิดการแจ้งเตือน**: คลิกที่ X
- **ดูรายละเอียด**: คลิกที่การแจ้งเตือน
- **ล้างทั้งหมด**: คลิก "Clear All"
